A PHONE NUMBER, SUCH AS (212) 767-8900, CAN BE THOUGHT OF AS HAVING THREE PARTS: E.G., THE AREA
CODE (212), THE EXCHANGE (767), AND THE NUMBER (8900). WRITE A PROGRAM THAT USES A STRUCTURE
TO STORE THESE THREE PARTS OF A PHONE NUMBER SEPARATELY. CALL THE STRUCTURE PHONE.
CREATE TWO STRUCTURE VARIABLES OF TYPE PHONE. INITIALIZE ONE, AND HAVE THE USER INPUT A NUMBER
FOR THE OTHER ONE. THEN DISPLAY BOTH NUMBERS.
